# Learn-AngularJs
### day01
1. 设计原则
    1.1 you aren't gonna need it.
        YANGI 不要写不需要的代码

    1.2 keep it simple and stupid.
        KISS 代码越简单越好

    1.3 Don't Repeat Yourself.
        DRY 善于封装、重用

    1.4 High Cohesion， Low Coupling.
        高内聚 低耦合

    1.5 single responsibility principle.
        SRP 单一责任原则

    1.6 open closed principle.
        OCP  开闭原则
2. 设计模式
23种设计模式，分三类:
    i. 创建型模式 共五种：工厂方法模式、抽象工厂模式、单例模式、建造者模式、原型模式。

    ii. 结构型模式，共七种：适配器模式、装饰器模式、代理模式、外观模式、桥接模式、组合模式、享元模式。

    iii. 行为型模式，共十一种：策略模式、模板方法模式、观察者模式、迭代子模式、责任链模式、命令模式、备忘录模式、状态模式、访问者模式、中介者模式、解释器模式。
